## Exam Analysis Skill 介绍

这是一个**全学段考试试卷深度分析与备考指导**的专业技能包，适用于从小学到高等教育的所有学科。核心功能是帮你对考试试卷进行专业级的逆向分析，提供精准的备考策略。

### 核心特点

1. **全学段覆盖**：支持中小学（K-12）到高等教育的所有学科
2. **教材精准锚定**：支持用户指定教材（作者+书名+版次）进行章节级精确对齐
3. **智能教材匹配**：未指定教材时自动识别学科并匹配主流教材（人教版/部编版等）
4. **三级知识点分解**：采用"章→节→具体知识点"的精确锚定格式
5. **系统化分析框架**：包含16个完整分析模块，从单卷分析到多卷交叉分析

### 主要功能模块

#### 单卷分析（11个必做模块）
1. **基础属性识别** - 学科、学段、考试类型识别
2. **知识点系统解构** - 三级知识点标注与教材锚定
3. **考试规范逆向工程** - 还原考试范围与能力层级
4. **难度定量评估** - 统一难度标准与系数计算
5. **学科核心素养分析** - 新课标核心素养对应分析
6. **试题情境与材料特征** - 情境化试题统计分析
7. **命题特点与陷阱设置** - 高频命题陷阱分析
8. **失分风险预测** - 模块化高频失分点整理
9. **单卷备考优先级** - 知识点复习优先级分类
10. **分析说明** - 局限性说明与基准依据
11. **自查与复核报告** - 迭代式自检循环（确保零缺陷）

#### 多卷交叉分析（5个模块）
12. **知识点频率与权重汇总** - 跨卷知识点统计与排名
13. **命题规律与难度概览** - 整体难度范围与命题模式
14. **总体复习方向** - 核心策略定位与时间分配
15. **核心复习重点** - 模块优先级与高频易错点
16. **分阶段具体复习内容计划** - 基础→强化→冲刺三阶段规划

### 使用流程

1. **试卷输入格式**：
```
【试卷1：试卷名称/来源】
(完整试卷内容)

【试卷2：试卷名称/来源】
(完整试卷内容)
```

2. **教材指定**（可选）：
- 格式：作者+书名+版次
- 示例：`邱关源《电路》第六版`
- 示例：`人教版《数学》八年级下册`

3. **执行顺序**：
   - 先完成所有单卷的11个模块分析
   - 再进行多卷交叉分析（模块12-16）
   - 每轮分析后必须进行迭代式自检，直到零缺陷

### 输出格式

- **默认格式**：Word (.docx) 文档
- **可选格式**：PDF
- **默认保存路径**：`D:\下载\`
- **文档结构**：包含封面、目录、完整分析内容
- **目录生成**：自动生成可更新的Word目录

### 技术特色

1. **严格的质量控制**：每轮分析后必须进行完整的自检循环，确保零缺陷输出
2. **统一的难度标准**：采用客观的难度分类标准（简单/中等/困难）和系数计算公式
3. **教材真实性验证**：所有章节锚定必须基于实际教材目录，严禁编造
4. **新课标对齐**：中小学试卷严格对应新课标核心素养框架
5. **情境分类细化**：包含社会热点情境、传统文化情境等细化分类

### 适用场景

- **教师**：分析试卷命题质量，优化教学重点
- **学生**：了解考试重点，制定针对性复习计划
- **教研人员**：进行试卷质量评估和命题规律研究
- **家长**：了解孩子考试情况，提供有效学习指导
