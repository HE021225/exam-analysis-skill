## 这是什么

`exam-analysis` 是一个**全学段试卷深度分析与备考指导**的专业技能包，覆盖从小学到高等教育的所有学科。给它一份（或多份）试卷原文，它会：

- 自动识别学科、学段、考试类型
- 逐题拆解知识点，并锚定到具体教材的「章 → 节 → 具体知识点」
- 量化评估难度（含难度系数计算），分析核心素养维度、命题陷阱与失分风险
- 单卷生成 **11 模块**全维度报告；多卷场景自动追加 **5 个**跨卷对比模块
- 输出专业 Word（.docx，默认）或 PDF 报告，可直接打印分发

它不是"帮你总结试卷"，而是一套**强约束、可复现**的命题逆向分析流程：每条结论都必须能在试卷原文里找到依据，每个知识点都必须锚定到真实教材章节，不允许凭印象编造。

## 适用场景

**K-12 全学段覆盖**：小学阶段各学科、初中阶段各学科、高中阶段各学科

**高等教育阶段**：专业课程考试、考研初试、考研复试、期末考试、期中考试、专业资格认证考试

## 核心特点

1. **全学段覆盖**：支持中小学（K-12）到高等教育的所有学科
2. **教材精准锚定**：支持用户指定教材（作者 + 书名 + 版次）进行章节级精确对齐
3. **智能教材匹配**：未指定教材时自动识别学科并匹配主流教材（人教版 / 部编版等）
4. **三级知识点分解**：采用「章 → 节 → 具体知识点」的精确锚定格式
5. **系统化分析框架**：包含 16 个完整分析模块，从单卷分析到多卷交叉分析
6. **强制迭代自检**：输出前反复执行 CHECK → FIX → RECHECK，确保零缺陷

## 教材锚定策略

| 优先级 | 策略 | 说明 |
|---|---|---|
| Priority 1 | 用户指定教材 | 格式：作者 +《书名》+ 版次，如 `邱关源《电路》第六版` |
| Priority 1.5 | 交互式询问 | 未指定教材时，识别学科 / 学段后主动询问用户 |
| Priority 2 | 自动匹配主流教材 | K-12 默认人教版 / 部编版，高等教育匹配国家规划教材 |
| Priority 3 | 通用知识结构兜底 | 标记为"暂定锚定"，建议后续补充教材信息 |

所有知识点统一按以下格式锚定：

```
[教材简称] 章 → 节 → 具体知识点
```

例如：

```
教材版本：人教版《数学》八年级下册
章：第十九章 一次函数
节：19.2 一次函数 → 19.2.2 一次函数的图像与性质
具体考点：一次函数图像中 k 值与图象经过象限的关系
```

若试卷跨越多本教材（如必修一 + 必修二 + 选择性必修一），会额外输出教材清单表，各教材章节独立编号、不做合并。

## 主要功能模块

### 单卷分析（11 个必做模块）
1. 基本属性识别（学科 / 学段 / 考试类型 / 题型分布）
2. 知识点体系解构（三级标注 + 教材锚定）
3. 考试规范逆向工程
4. 难度定量评估
5. 学科核心素养与能力维度分析
6. 题目情境与材料特征分析
7. 命题特征与陷阱设置分析
8. 失分风险预测
9. 单卷备考优先级与复习建议
10. 分析说明（局限性 + 教材版本声明）
11. 自查与复核报告（触发下方自检闭环）

### 多卷交叉分析（多份试卷时自动触发，5 个模块）
12. 跨卷知识点频率与权重汇总
13. 跨卷命题规律与难度概览
14. 总体复习方向
15. 核心复习重点
16. 分阶段具体复习计划（基础巩固 → 强化提升 → 冲刺回顾）

模块顺序不可跳过、不可合并、不可增删；每条结论都必须能追溯到具体题号。

### 自检闭环

单卷模块 11 完成后不会直接输出，Skill 会执行：

```
① 全维度自检（模块完整性、教材锚定真实性、难度标准一致性、
   分数计算、结论与原文对应关系……逐项排查）
② 记录本轮发现的全部缺陷
③ 修复全部缺陷 → ④ 从头重新执行①
⑤ 重复 ①-④，直至某一轮结果为"零缺陷"才允许输出
```

> v3.1 起，是否触发这一轮自检改为可由用户手动选择，此前版本为强制 AI 自检。

## 核心原则

1. **严格模块顺序**：不跳过、不合并、不自增自减模块
2. **证据驱动**：所有结论必须有试卷原文支撑，标注对应题号
3. **不编造**：教材章节名称必须来自真实教材目录，不可凭记忆杜撰
4. **迭代自查**：CHECK → FIX → RECHECK 循环，直到零缺陷才输出
5. **统一标准**：多卷场景下知识点命名、难度标准、教材版本完全统一

## 难度评估标准

| 难度等级 | 特征 | 预估得分率 |
|---|---|---|
| 简单 | 单一知识点，直接套用公式 / 概念，无陷阱 | 0.85 |
| 中等 | 2-3 个关联知识点，需 1-2 步推理，含常见易错点 | 0.60 |
| 困难 | 跨模块综合，多步推导 / 建模，题型新颖、开放度高 | 0.30 |

难度系数公式：
```
难度系数 = (简单题分值 × 0.85 + 中等题分值 × 0.6 + 困难题分值 × 0.3) ÷ 总分
```

| 难度系数 | 含义 |
|---|---|
| 0.80–1.00 | 非常容易 |
| 0.65–0.79 | 偏易，适合基础评估 |
| 0.50–0.64 | 中等，区分度均衡 |
| 0.35–0.49 | 偏难，区分度好 |
| < 0.35 | 非常困难，选拔性强 |

## 使用流程

**1. 基本用法**：直接发送试卷内容

```
帮我分析这份试卷：

【试卷1：2026年春季《电路》期末考试】
一、选择题（每题2分，共20分）
1. 图示电路中，已知 R=10Ω，电压源 Us=20V，则电流 I 为（ ）
   A. 0.5A   B. 1A   C. 2A   D. 4A
...
```

**2. 指定教材版本（推荐）**：附带教材信息可获得精准章节锚定

```
帮我分析这份试卷，教材是 邱关源《电路》第六版：

【试卷1：电路期末考试】
...
```

**3. 多卷交叉分析**：用 `【试卷N：名称】` 依次分隔多份试卷

```
帮我分析以下三份试卷，教材是人教版《数学》八年级下册：

【试卷1：2024年月考】
...
【试卷2：2024年期中】
...
【试卷3：2024年期末】
...
```
Skill 会先独立完成每份试卷的 11 模块分析，再自动进入跨卷综合分析。

**4. 确认是否复检（v3.1 起）**：单卷分析完成后，Skill 会询问是否执行自检闭环，你可以选择让 AI 自动复检至零缺陷，也可以选择跳过、直接输出当前结果。

## 输出格式

- **默认格式**：Word（.docx），兼容性最好，包含封面、目录、统一字体与表格样式
- **可选格式**：PDF，固定排版，适合打印与分发
- **文件命名**：`[学科]-[考试类型]-试卷分析报告-[日期].docx`
- **默认保存路径**：`D:\下载\`，生成前会与你确认保存位置，支持自定义

## 典型场景

| 场景 | 用法 |
|---|---|
| 考后复盘 | 上传刚考完的试卷，获取逐题拆解 + 失分预测 |
| 考前冲刺 | 上传 3-5 份历年真题，获取跨卷高频考点 + 分阶段复习计划 |
| 教学研究 | 分析命题规律、陷阱设置模式、核心素养分布 |
| 家长辅导 | 孩子考完试后用报告精准定位薄弱知识点 |
| 考研备考 | 上传目标院校历年专业课真题，锚定到指定教材章节 |

## 注意事项

- 试卷内容需完整输入（含题目、选项、分值），越完整分析越精准
- 高等教育试卷建议附带教材信息，以获得章节级精度；教材版本给错会直接导致章节锚定错误
- 单份试卷的结论有局限性，多卷交叉验证的结果更可靠
- 暂不支持图片格式试卷，可先 OCR 转文字后再使用
- 默认输出路径面向 Windows 场景，其他系统可在确认保存位置时自行指定
